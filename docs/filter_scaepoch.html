scaepoch

Milestone: 1

Synopsis

Extracts epoch time from designated field 

filter {
  scacsv {
    input_field => ... # string (required)
    output_field => ... # string (required)
    pattern => ... # string (required)
  }
}

Details

Converts value in input_field to an epoch time, using the pattern specified in pattern and the resulting epoch time is placed in output_field.

For example, a timestamp like "19/12/14 12:04 AM" might be decoded using the following 
    scaepoch {
      input_field =>  "Tiempo de muestra"
      output_field => "timestamp"
      pattern => "%d/%m/%y %l:%M %p"
    }
A typical use of these timestamps is to assign start/end times to files output by scacsv

input_field

  • Value type is String
  • There is no default for this setting

Name of field where string representing timestamp is found

output_field

  • Value type is String
  • There is no default for this setting

Name of field where resulting timestamp value is placed. very often this is simply "timestamp"

pattern

  • Value type is string
  • There is no default for this setting

Name of field where an format string used to decode the timestamp is found. The pattern format is Ruby strftime format defined here here
